1. Inicio
2. Definir el archivo HTML con la declaración `<!DOCTYPE html>`
3. Abrir la etiqueta `<html>`
4. Abrir la etiqueta `<body>`
5. Mostrar un encabezado de nivel 1 `<h1>` con el texto "JavaScript Variables"
6. Mostrar un párrafo `<p>` con el texto "In this example, x, y, and z are variables."
7. Mostrar un párrafo `<p>` con un `id="demo"`, que será utilizado para mostrar el resultado de la operación.
8. Abrir la etiqueta `<script>` para incluir código JavaScript
9. Definir una variable `x` con el valor 5
10. Definir una variable `y` con el valor 6
11. Sumar las variables `x` e `y` y almacenar el resultado en la variable `z`
12. Utilizar `document.getElementById("demo").innerHTML` para seleccionar el elemento con `id="demo"` y actualizar su contenido con el valor de `z`
13. Cerrar la etiqueta `<script>`
14. Cerrar la etiqueta `<body>`
15. Cerrar la etiqueta `<html>`
16. Fin
